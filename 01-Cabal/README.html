Voit buildata projektin sanomalla:
> cabal build

tai suorittaa projektin executablen:
> cabal run
> cabal run tervehdys

tai ajaa testit
> cabal test

tai configuroida flägien kanssa
> cabal configure -fWithOpenGL

tai käynnistää REPL:n projektin riippuvuuksilla:
> cabal repl
mikä käynnistää projektin ensimmäisen "osan", oli se kirjasto tai executable.
Voit antaa nimen perässä jos haluat valita:
> cabal repl tervehdys
GHCJS-projektissa tämä itseasiassa käynnistää REPL:n joka pyörii selaimessa!

Buildaamisen jälkeen kaikki executablet ovat ajettavissa:
> dist/build/moikkahdus/moikkahdus
>> "Moikka kaikille!"
> dist/build/tervehdys/tervehdys
>> "Hello Digital Ocean!"
