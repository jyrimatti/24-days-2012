Lisäsin cabal-tiedostoon seuraavan rivin jotta sai GC-statistiikat näkyviin:
GHC-Options: -with-rtsopts=-T

Selaimeen saa helposti mukaan omiakin käyriä.

Mutta koska tuo selain-UI on vain pikkukilke, niin arvot saa pihalle myös JSON:na jotta niitä voi seurata millä tahansa työkalulla.

Käynnistä server pyörimään yhteen shelliin:
> cabal run notify
ja lisäile lahjoja toisesta shellistä:
> cabal run present "leegoja"
ja näet UI:sta miten notifikaatiot lisääntyvät.
