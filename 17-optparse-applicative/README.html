optparse-applicative on aika näppärä.
Komentoriviparametrien käsittelyn pystyy tekemään helposti, erotellen tietorakenteet, parserit ja itse toiminnallisuuden toisistaan.
Tukee mm. tyyppimuunnoksia (read), vapaavalintaisuutta, argumentit, lyhyet/pitkät valinnat, help-tekstit, command-hierarkia...

En toteuttanut kaikkia valintoja, mutta kokeile ajaa vaikkapa:
> cabal run -- greet Solita

Help-tekstit saa helposti hierarkisesti eri commandeille. Kokeile esim:
> cabal run -- --help
> cabal run -- greet --help
> cabal run -- haistatus --help
